# Tkinter_Project

# ⏰ Clock, Calendar & Alarm Watch using Tkinter

A simple desktop application built using Python and Tkinter that provides a Digital Clock, Calendar, and Alarm functionality in a single interface. This project demonstrates the use of GUI development in Python.

---

## 📌 Features

- 🕒 Real-Time Digital Clock
- 📅 Interactive Calendar
- ⏰ Alarm Setting Functionality
- 🎨 User-Friendly Graphical Interface
- 🔔 Alarm Notification with Sound
- 🖥️ Lightweight and Easy to Use

---

## 🛠️ Technologies Used

- Python
- Tkinter
- Datetime Module
- Time Module
- Threading Module
- Calendar Module

---

## 🚀 Getting Started

### Prerequisites

Make sure Python is installed on your system.

- Python 3.x
