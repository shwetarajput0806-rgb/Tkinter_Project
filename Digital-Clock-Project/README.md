# ⏰ Digital Watch (Tkinter Project)

## 📌 Description

This project is a simple **Digital Watch application** built using Python and Tkinter.
It displays the current time in a digital format and updates automatically every second.

---

## 🚀 Features

* Real-time digital clock
* Automatic time update (every second)
* Simple and user-friendly interface
* Lightweight application

---

## 🛠️ Technologies Used

* Python
* Tkinter

---

## 📂 Project Structure

DigitalWatch/
│── main.py
│── README.md

---

## ▶️ How to Run

1. Make sure Python is installed on your system
2. Download or clone this project
3. Open terminal / command prompt
4. Run the file:

python main.py

---

## 💡 Working Principle

* Tkinter is used to create the graphical interface
* The `time` module fetches the current system time
* The `after()` method updates the time every 1000 milliseconds
