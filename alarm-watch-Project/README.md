# ⏰ Alarm Watch Project (Tkinter)

## 📖 Overview
This is a simple **Alarm Watch application** built using Python and Tkinter.  
It allows users to set alarms and get notifications at the specified time.

---

## 🚀 Features
- Set custom alarm time ⏰  
- Easy-to-use GUI using Tkinter 🎨  
- Real-time clock display  
- Alarm notification sound 🔔  
- Lightweight and fast application  

---

## 🛠️ Tech Stack
- Python 🐍  
- Tkinter (GUI Library)  
- Time Module
