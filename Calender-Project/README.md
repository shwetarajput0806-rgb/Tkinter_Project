
# 📅 Calendar Application using Tkinter

A simple Calendar Application built using Python and Tkinter. This desktop application allows users to view calendars for any month and year through an interactive graphical interface. The project demonstrates GUI development and date handling in Python.

---

## 📌 Features

- 📅 Display calendar for any month and year
- 🎨 User-friendly graphical interface
- 🖥️ Lightweight and easy to use
- ⚡ Fast and responsive application
- 📆 Interactive calendar view

---

## 🛠️ Technologies Used

- Python
- Tkinter
- Calendar Module

---

## 🎯 Learning Outcomes

- GUI development using Tkinter
- Event-driven programming
- Working with dates and calendars in Python
- Desktop application development
